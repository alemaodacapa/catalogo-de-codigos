# 📚 Code Academy - Catálogo de Códigos

Bem-vindo ao **Code Academy**, um catálogo interativo de trechos de código com categorias organizadas por linguagem. Esta aplicação web foi criada para ajudar desenvolvedores a encontrar rapidamente exemplos de código úteis e reutilizáveis, com destaque de sintaxe automático e sistema de favoritos.

## 🚀 Funcionalidades

- 🔍 **Busca por código** usando palavras-chave
- 📁 **Categorias organizadas** por linguagem:
  - HTML
  - CSS
  - JavaScript
  - PHP
  - Twig
  - E-commerce API
- ⭐ **Sistema de favoritos** para salvar snippets úteis
- 🎨 **Interface moderna** e responsiva
- 💡 **Highlight automático** de sintaxe com [highlight.js](https://highlightjs.org/)

## 🛠️ Tecnologias Utilizadas

- HTML5
- CSS3
- JavaScript (ES Modules)
- [highlight.js](https://cdnjs.com/libraries/highlight.js)
- Estrutura modular e limpa com `app.js` e `styles.css`

## 🧪 Como Usar

1. Clone este repositório:

```bash
git clone https://github.com/alemaodacapa/catalogo-de-codigos.git
```

2. Abra o arquivo `index.html` em seu navegador.

3. Pesquise um trecho de código ou navegue pelas categorias disponíveis.

4. Clique em ⭐ **Salvar nos Favoritos** para manter seus trechos preferidos salvos.
